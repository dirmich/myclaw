# MyClaw (マイクロー) 🤖

MyClawは、リモートサーバー上で**OpenClaw**を簡単にインストールおよび管理できるように設計された、ウェブベースのインストールマネージャーです。 SSHを介してサーバーに接続し、DockerベースのOpenClawゲートウェイを自動的に構成および展開します。

[English](./README.md) | [한국어](./README-ko.md) | [简体中文](./README-cn.md) | [Español](./README-es.md)

## ✨ 主な機能

- **リモートSSHインストール**: 複雑なターミナルコマンドなしで、ウェブUIからサーバー情報を入力するだけで簡単にインストールできます。
- **Dockerベースの展開**: DockerおよびDocker Composeを自動的に構成し、一貫した実行環境を保証します。
- **自動構成の最適化**: `openclaw.json`を自動的に生成し、Telegramチャネルのスキーマエラーを防止するための最適な設定を適用します。
- **統合されたセキュリティ認証**: ダッシュボードアクセスのための`gatewayToken`の自動生成と認証フローをサポートします。
- **リアルタイムログモニタリング**: インストールプロセスをリアルタイムで監視し、問題が発生した場合はログを分析できます。

## 🚀 はじめに

### 事前準備
- ローカル環境: [Bun](https://bun.sh) または Node.js (v18以上)
- リモートサーバー: SSHアクセスが可能なLinuxサーバー（Ubuntu推奨）およびSudo権限

### インストールと実行

1. **リポジトリをクローンする**
   ```bash
   git clone https://github.com/dirmich/myclaw.git
   cd myclaw
   ```

2. **依存関係をインストールする**
   ```bash
   bun install
   # または npm install
   ```

3. **開発サーバーを実行する**
   ```bash
   bun dev
   ```

4. **アクセス**
   ブラウザで `http://localhost:3000` を開きます。

## 🛠 使用方法

1. **サーバー情報の入力**: リモートサーバーのIP、ポート、ユーザー名、パスワード（またはSSHキー）を入力します。
2. **AI設定**: 使用するAIモデル（OpenAI、Anthropicなど）のAPIキーを入力します。
3. **Telegram設定（オプション）**: Telegram Bot Tokenを入力すると、即時通信できるように自動構成されます。
4. **インストール開始**: 「インストール」ボタンをクリックすると、リモートサーバーでのDockerインストールからOpenClawの実行まで自動的に進行します。
5. **ダッシュボードへのアクセス**: インストールが完了したら、提供されたリンクを使用してOpenClawコントロール画面に直接移動できます。

## ☁️ デプロイガイド (GitHub Pagesに関する情報)

> [!WARNING]
> **MyClawはGitHub Pagesで直接実行することはできません。**
> 
> GitHub Pagesは **静的（Static）** ファイルのみをホスティングできます。 MyClawはSSH接続およびリモート制御のために **Node.jsバックエンド（API Routes）** を必要とします。 そのため、以下のようなプラットフォーム経由でのデプロイを推奨します：
> - **Vercel**: Next.jsプロジェクトを最も簡単にデプロイする方法です。
> - **プライベートVPS**: Dockerなどを使用して、ご自身のサーバーに直接ホスティングできます。

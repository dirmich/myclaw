https://www.simpleclaw.com/ 이것처럼 
https://github.com/openclaw/openclaw를 쉽게 설치할 수 있는
web ui를 만들고 싶어.
virtualbox나 aws ec2, macmini에 설치할 수 있도록 ssh접속정보를 
입력받고, ai나 telegram키를 입력받아서 자동으로 설치해서 쉽게
실행할 수 있도록 해줘.
ai키나 telegram키는 입력하면 연결테스트해서 연결되는지 확인하고 넘어가게. nextjs, tailwindcss, shadcn ui, bun.sh를 사용해서 만들어줘. 
../openclaw에 openclaw를 clone해놨어.
설치는 최대한 간단하게 할 수 있게하고, 언어는 영어기본에 한/일/중/스페인어를 선택할 수 있도록 해주고.
https://www.simpleclaw.com/의 ui를 참고해서 최대한 간단하고 쉽게 설치할 수 있도록 해줘.
